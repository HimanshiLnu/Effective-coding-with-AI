# PROJECT_CONTEXT.md

## Project
This is a student project for AIM 5012: Effective Coding with AI.

## Current goal
Build a simple portfolio website that is working, tested, and explainable.

## Main files
- `index.html`: page structure and content
- `style.css`: visual styling and responsive layout
- `PROMPTS.md`: AI use log and reflection
- `README.md`: project overview and how to run/check the project

## Commands
- Open locally: open `index.html` in a browser
- Optional local server: `python3 -m http.server 8000`
- Check Git status: `git status`
- Review changes: `git diff`

## Course workflow
1. Inspect first.
2. Plan before editing.
3. Make one small change.
4. Test visibly.
5. Explain the change.
6. Log AI help in `PROMPTS.md`.
