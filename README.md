# Week 1 Tool Setup

This project is for AIM 5012: Effective Coding with AI.

## Project Goal

The goal is to build a simple portfolio webpage that is working, tested, and explainable.

## Files

- `index.html`: webpage structure and content
- `style.css`: webpage styling and responsive layout
- `PROMPTS.md`: AI use log and reflection
- `README.md`: project overview

## How to Open

Open `index.html` in a browser.

## What I Practiced

- Inspecting code before editing
- Planning before making changes
- Editing HTML and CSS
- Testing in the browser
- Recording AI use in `PROMPTS.md`
